# Sistema de Controle de Ambientes do SENAI

#### Esse sistema foi desenvolvido pelos alunos do 3º ano do Ensino Médio, no itinerário de Desenvolvimento de Sistemas, com o objetivo de gerenciar os empréstimos de patrimônio da instituição.<br><br>

# Features:
#### — Manipulação de ambientes ativos e inativos;
#### — Requisição de empréstimo de equipamento;
#### — Solicitação/aprovação por parte de admin;
#### — Persistência de dados em banco estruturado;
#### — Exportação em formato .xml;
#### — Integração com planilha corporativa online;
#### — Interface amigável e fácil de aprender.<br><br>
